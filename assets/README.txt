Put photo files here (PI photo, member photos) — e.g. assets/torikul-islam.jpg

Then, in the Member Portal admin panel, paste the path "assets/torikul-islam.jpg"
into the relevant "Photo path or URL" field (PI profile, or a member's entry).
A full https:// URL to an image hosted elsewhere works too.

If no photo is set, the site automatically falls back to a colored
initials avatar — you don't need to add a photo right away.
